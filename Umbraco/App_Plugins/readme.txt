Diplo Trace Log Viewer for Umbraco 7

"Diplo Trace Log Viewer is a Developer tree section in Umbraco 7 that allows you to easily select and view the contents of the trace logs that the Umbraco CMS generates."

By Dan Booth 2014-2017 (with grateful help from Sebastiaan Janssen, Jeavon Leopold, Shannon Deminick, Bjarne Fyrstenborg and Umbraco community)

Package - http://our.umbraco.org/projects/developer-tools/diplo-trace-log-viewer

Nuget - https://www.nuget.org/packages/DiploTraceLogViewer/

Source - https://github.com/DanDiplo/UmbracoTraceLogViewer

Blog - http://www.diplo.co.uk/blog/web-development/diplo-trace-log-viewer-for-umbraco/

Uses Angular SmartTable - http://lorenzofox3.github.io/smart-table-website/
